Dynamic Pathfinding Visualizer
Overview

A real-time pathfinding visualizer using Python and Pygame.
It shows how different algorithms explore a grid to find a path from start to goal.

Algorithms:

A* (A-Star)

Greedy Best-First Search (GBFS)

Heuristics:

Manhattan Distance

Euclidean Distance

Installation

Install Python 3.x

Install Pygame:

pip install pygame

Run the program:

python pathfinding_visualizer.py
Controls

Left Click: Draw walls

R: Reset grid

M: Random walls

SPACE: Start search

1: Switch to A*

2: Switch to GBFS

H: Toggle heuristic

Close Window: Quit

Features

Step-by-step visualization of node exploration

Shows visited nodes, frontier, and final path

Displays path cost, nodes visited, and execution time

Dynamic wall creation and random obstacles# Dynamic-Pathfinding-Visualizer
